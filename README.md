3️⃣ What is the difference between map(), filter(), and forEach()?
The map() method creates a new array by transforming every element of the original array

The filter() method creates a new array by selecting elements that meet a specific condition. It returns only the elements that match the criteria, leaving the original array unchanged

The forEach() is a loop method that accesses each element of an array one by one and executes a specific function for every element

5️⃣ What are template literals?
Template literals are a modern way to handle string values in JavaScript using backticks (`). They allow embedded expressions through ${} and support multi-line strings easily, making the code cleaner and more readable
